Course Curriculum: HTML Basics

Objectives

  1. To design a simple course curriculum webpage using HTML.
  2. To practice the use of basic text formatting tags (bold, italic, underline, special characters).
  3. To learn the implementation of ordered, unordered, nested, and mixed lists.
  4. To understand webpage content structuring using headings and paragraphs.

Tools Required

  • Computer / Laptop with any OS
  • Text Editor (VS Code, Sublime Text, Notepad++)
  • Web Browser (Chrome, Firefox, Edge)

Theory

1. Headings

HTML provides six levels of headings: <h1> to <h6>, used to display titles and subheadings in different sizes.

2. Paragraphs

Use the <p> tag to define blocks of text content.

3. Formatting Tags

  • <b> for bold text
  • <i> for italic text
  • <u> for underlined text
  • Special characters: &copy; ©, &gt; >, &lt; <
